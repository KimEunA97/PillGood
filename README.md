# PillGood
사용자 친화적인 건강 알리미 서비스

---

## 서비스 소개 및 목표
- 약에 대한 정보 제공 및 앱 내 페이지에 등록
- 영양제나 비타민 복용시간 알림
- 위젯을 이용해 간편한 시간 체크
- 복용 기록을 통해 의사와 자세한 진단 가능

---

## 사용 기술 및 데이터
- Expo
- 공공데이터 API(e약은요)

---

## 기능 설명
1. 약을 검색하고 선택한다.
2. 복용할 시간을 설정하면 제시간에 알림을 받는다.
3. 푸시 알림이 오면 화면 상단 버튼을 통해 완료한다.
4. 홈 화면에 위젯을 추가하면 확인 버튼을 누를 수 있다.

---

## 현재까지 완료된 작업

### *front*
- 검색 기능 모달창
- 원하는 약 검색 및 취소 기능
- 결과 개수에 따른 선택 리스트 모달창 제공
- 선택한 약을 페이지 내 등록
### *back*
- api 파라미터 요청
- 결과값에 적합한 데이터 가공
